## 1. 命令（确切一条 + 等价路径）

| 用途 | 命令 |
|---|---|
| **主命令（推荐）** | `python -m fulltext_fetcher -f a5_smoke_input.txt --institutional --route-b cf-only --email you@uni.edu` |
| 一键路径（读到 `enabled` JSON / `FTF_INSTITUTIONAL=1` 自动进机构模式，可省 `--institutional`） | `python run_all.py -f a5_smoke_input.txt --route-b cf-only` |

## 3. 凭据放哪个本地文件（不入库）

**唯一落点：仓根 `.ftf_institutional.local.json`**（由 `.ftf_institutional.local.json.example` 复制而来）。

```powershell
Copy-Item .ftf_institutional.local.json.example .ftf_institutional.local.json
```

字段（照《接入指引-141》§2 填）：

| 字段 | 必填 | 说明 / 示例 |
|---|---|---|
| `enabled` | ✅ | **必须 `true` 才生效** |
| `provider` | ✅ | `ezproxy` / `shibboleth` / `carsi` / `webvpn` / `openathens` / `manual` |
| `ezproxy_prefix` | 视机制 | EZproxy 前缀或代理裸域；非 EZproxy 留 `null` |
| `institution_cookie` | ✅ | 浏览器登录后用 Cookie-Editor 导出的 Header String（`k1=v1; k2=v2`） |
| `institution_domains` | 建议 | 只对这些出版商域启用，如 `["sciencedirect.com","pubs.acs.org","pubs.rsc.org"]` |
| `cookie_store_path` | 可选 | 默认 `.ftf_cookies.local.json`（持久化 Cookie 仓，同样不入库） |

**不落文件的等价法（CI/一次性）**：设 `FTF_INSTITUTIONAL=1` + `FTF_EZPROXY_PREFIX` / `FTF_INSTITUTION_COOKIE` / `FTF_INSTITUTION_DOMAINS`（框架层 `FTF_*`，核心 CLI 与 `run_all.py` 均识别）。

**安全（已在 `.gitignore` 保障）**：`.gitignore` L44 `.ftf_*.local.json` 已忽略本地凭据/Cookie 文件；框架 `InstitutionalCredentials.__repr__` 自动脱敏，日志不落明文。仍请 `git status` 自查。

## 5. 排障速查

| 现象 | 处理 |
|---|---|
| 大面积 401/403 | Cookie 过期 / 未覆盖该域 → 重登导 Cookie；确认 `institution_domains` 含该站 |
| 强 CF 站仍 miss | 未开 route-B → 加 `--route-b cf-only`（需 nodriver + 有头显示） |
| 机构模式没生效 | 查 JSON `enabled:true` / `FTF_INSTITUTIONAL=1`；看日志是否打印“机构凭据自 FTF 加载” |
| route-B 静默 no-op | 缺 nodriver / 无显示环境 → 装 nodriver，或用有头 / xvfb 虚拟显示 |
