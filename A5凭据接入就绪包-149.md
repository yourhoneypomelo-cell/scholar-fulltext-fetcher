## ② 凭据接入指引（精简版 · 全量见 `A5用户凭据接入指引-141.md`）

### 支持的接入方式

| 机制 | 你要准备 | 现在能用吗 |
|---|---|---|
| **EZproxy（前缀式/主机名改写式）** | 登录后 Cookie 串 + 前缀 `https://login.ezproxy.你校.edu/login?url=` 或裸域 `ezproxy.你校.edu` | ✅ 立即可用（HTTP 直链 + **可见浏览器自动登录已布线**，见 §③） |
| **Shibboleth / CARSI / WebVPN / OpenAthens** | 浏览器完成 SSO 后，导出目标出版商域 Cookie 串 | ✅ **手动导 Cookie 即可用**；自动可见浏览器登录分支待 -153 |

### 三步启动

1. **浏览器登录**机构入口（EZproxy/SSO）到能看全文（地址栏脱离 `login/cas/sso/idp/wayf`）→ 用 **Cookie-Editor → Export → Header String** 导出 Cookie 串。
2. **写入凭据**（二选一）：
   - 本地 JSON（推荐）：`Copy-Item .ftf_institutional.local.json.example .ftf_institutional.local.json`，填 `enabled=true / provider / ezproxy_prefix / institution_cookie / institution_domains`。
   - 环境变量：`FTF_INSTITUTIONAL=1` / `FTF_EZPROXY_PREFIX` / `FTF_INSTITUTION_COOKIE` / `FTF_INSTITUTION_DOMAINS`。
3. **一键运行**（凭据到手后）：

```powershell
# 先冒烟本就绪包的 18 条测试集
python -m fulltext_fetcher -f out/_a5_testset_176.txt --institutional --route-b cf-only --email you@uni.edu
# run_all 一键路径:读到 enabled 的本地 JSON / FTF_INSTITUTIONAL=1 即自动进机构模式
python run_all.py -f out/_a5_testset_176.txt --route-b cf-only
```

### 安全红线（务必遵守）

- ❌ 严禁把 Cookie/前缀**硬编码**进任何 `.py`；❌ 严禁贴进 **chat / run.log / metadata.jsonl / 任何产物**；❌ 严禁 `git add` 本地凭据文件。
- ✅ 凭据**只本地**：`.gitignore` 已忽略 `.ftf_*.local.json`（模板 `.example` 可入库，仍请 `git status` 自查）。
- ✅ 框架自动脱敏：`InstitutionalCredentials.__repr__` 只显示 `abcd…yz`，日志无明文；无订阅时直链回 401/403 且被 `%PDF` 魔数校验过滤，不污染产物。
- ✅ 仅对**本人合法有访问权**的资源、经正门等效人工登录取用；默认关、低并发（放量用 `-c 2 --per-host-interval 1.0` 防 bulk 风控）。
