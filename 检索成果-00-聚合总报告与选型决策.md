## 一、一页结论（TL;DR）

1. **能走正门就别翻墙**：Google 从无官方 Scholar API、`robots.txt` 禁止自动化抓取，任何"过人机"都是灰色 + 永无止境的军备竞赛。**首选主线＝角度2 官方开放 API（OpenAlex / Semantic Scholar / Crossref / Unpaywall）——根本不存在验证码**。
2. **唯一非走 Scholar 不可的，是 Scholar 原生指标**（其口径的被引/版本/作者档）。这时两条路：
   - **省心付费**：角度4 商业 / 第三方 Scholar API（SerpApi 等）——把过人机整包外包；
   - **自建斗法**：角度1（GitHub 现成载体）＋ 角度3（分层反爬技术）＋ 角度6（代理基础设施）组合。
3. **人工 / 低量**：角度7 镜像站（把反爬转嫁运营方）；**免代码即用**：角度5 工具（Publish or Perish / Zotero / Consensus·Elicit·scite 等，多数走开放数据、不碰 GS 反爬，合规优于镜像站）。
4. **付费墙内全文谁都拿不到**——这是合理边界，靠机构订阅/馆际互借解决，不在本课题范围内强求。
5. **方法论铁律**（角度3）：先判断"被哪一层拦（L1 TLS / L2 JS / L3 自动化协议 / 翼A IP / 翼B 行为）"，再加**最便宜的对应工具**，别无脑堆栈。

- **第一优先**：角度2。覆盖 OpenAlex 4.8 亿 + S2 2 亿，发现/元数据/被引网络已基本对标 Scholar，OA 全文经 Unpaywall/OpenAlex 直链；大规模直接落官方 snapshot 本地库（彻底无限额、无封禁、无验证码）。

## 六、各角度核心结论摘要

**角度1（GitHub 载体）**：GitHub 项目绝大多数只做"驱动浏览器+解析页面+串下载源"，**真正的过人机靠外接**（代理/SerpApi/打码）。元数据底座首推 `scholarly`（事实标准，须配代理）；批量下载首推 `PyPaperBot`（已复活）；多源聚合 `paperscraper`（2026-06 最活跃、反爬暴露面低）；省心规模化用 `serpapi` 客户端 / `resp`。clone 前必看 last commit 与 issues（`scholar.py` 2022 停更、`scihub.py` 停滞）。

**角度2（开放 API·主线）**：Crossref（**1.8 亿+** DOI，免费无 key）/ OpenAlex（4.8 亿 works；2026 起 **key 实质必需**——无 key 仅 ~$0.10/天 demo、`mailto` 礼貌池已废为 key-only，每 key 送 $1/天，snapshot CC0；注：API 默认 `/works` 实测计数约 3.18 亿，差额疑为 2025-11 新增未全量入默认查询）/ Semantic Scholar（2 亿，bulk 检索 + TLDR）/ Unpaywall（4000 万 OA 全文，仅需**真实** email、10 万/天）。**无任何验证码、官方鼓励调用**。唯一短板＝付费墙内全文拿不到。**强烈建议作为工作组首选主线**。〔98 复核并入：Crossref 实测 1.84 亿、OpenAlex key-only〕

**角度3（反爬技术·备线）**：2026「三层+两翼」检测模型——L1 TLS/HTTP2 指纹（`curl_cffi`/`tls-client`）、L2 JS 指纹（`Patchright`/`Camoufox`）、L3 自动化协议（**`nodriver`，benchmark 唯一 31 目标零封锁**）、翼A IP（住宅/移动代理）、翼B 行为（打码兜底 CapSolver/CapMonster）。铁律：**先定位被拦层，再加最便宜工具**；代理≠万能（L1/L2 指纹仍来自真机）。

**角度4（商业服务·买断免战）**：12+ 家——专用 Scholar API（SerpApi 字段最全+US Legal Shield+SOC2；Scrapeless ~$0.8/1k 便宜；SearchApi 中端；ScaleSERP/ScrapingBee）＋ 通用解锁/SERP（Bright Data/Oxylabs/ScraperAPI/ZenRows/DataForSEO/SearchCans $0.56/1k）＋ Apify 聚合 Actor。**把过人机整包外包，是除自建外唯一稳拿 Scholar 原生字段的工程化路线**；代价是持续付费、合规仍灰色（服务商法律盾只缓释不免责）。

**角度5（即用型工具 & 平台·免代码）**：面向不写代码的科研用户。**桌面**＝Publish or Perish（免费，v8.19，8 数据源、GS 被引/h 指数计量）；**攒库/取全文**＝Zotero+Connector（一键存元数据+PDF）、EndNote Click(原 Kopernio)/Unpaywall/Open Access Button（一键找 OA 全文）；**AI 学术搜索（网页即用）**＝Consensus（循证问答，免费+$15/mo）、Elicit（PRISMA 系统综述+数据抽取，免费+$49/mo）、scite（Smart Citations 支持/反驳判别，$20/mo）、ResearchRabbit（免费引文图谱）、Connected Papers（$3/mo 相似度图谱）。**核心洞察：这些工具大多根本不碰 GS 反爬**——AI 平台/OA 扩展走 S2/OpenAlex/Unpaywall 开放数据（≈角度2 无代码化）、Zotero/PoP 把抓取降为「人工浏览一键存」，故在实践中绕开人机认证且合规安全（人工低量优于角度7 镜像站）。

**角度6（代理基础设施）**：抓 Scholar 被封首要在「翼A：IP 信誉」。**类型选型**：数据中心（易封·不推荐）< 住宅（主力·每 5–15 次换 IP）< ISP 静态（养号/持久会话）< 移动 4G/5G（最难封·最贵·难目标兜底）。**七大供应商 2026 住宅 $/GB 核验**：Bright Data（150M+，$5.04→议价 <$3）、Oxylabs（175M+，$6→$2.5）、Decodo（原 Smartproxy，125M+，$3.75→$2）、SOAX（155M+，$3.6→$0.32）、NetNut（85M+，ISP 强项，$3.53→$1.59）、IPRoyal（$7→$1.75，流量不过期）、Webshare（$1.4 起 + 免费档）。**开源自建代理池** jhao104/proxy_pool(23.4k)/ProxyBroker(4.2k)/proxifly(5.8k) 等——但免费代理抓 Scholar 基本不可行，仅学机制/兜底。**核心结论：代理是自建路线的「必要非充分」资源，必须与角度3 的 L1/L2/L3 指纹对抗组合，单换 IP 无效**；与角度4 的边界＝角度4 买整包服务、角度6 只买 IP 这一层（同一批厂商两边都出现）。

**角度7（中文社区与镜像站）**：镜像站（烂番薯学术等）把反爬转嫁运营方，适合人工/低量；标准链路"查（烂番薯）→下（Sci-Hub 凭 DOI）→管（Zotero）"。风险：篡改/恶意/广告，需核对元数据一致、站内不登录。**不便程序化批量**，建议另走角度2/4。

## 七、合规与风险总览

1. **法律性质**：直抓 Scholar 违反 ToS 与 `robots.txt`，全程灰色（角度1/3/4/6/7）；唯**角度2 开放 API 是白色合规**。
2. **服务商法律盾≠免责**：SerpApi US Legal Shield、SearchApi Legal Protection、ScrapingBee GDPR/CCPA 只**部分转移/缓释**风险（角度4）。
3. **军备竞赛成本**：自建栈随 Chrome/reCAPTCHA/Cloudflare 升级失效（UC→nodriver 即前车之鉴），维护人力高（角度3）。
4. **数据正确性**：HTML 解析随改版字段易缺；Sci-Hub 内容可能不符——下载后务必核对元数据一致（角度1/7）。
5. **计费暗坑**：Scrapeless 余额不滚存、ZenRows credit 倍率、Apify 2026-10 计费切换——按"有效单价"测算（角度4）。
6. **付费墙边界**：所有路线都拿不到付费墙内全文，靠机构订阅解决。

## 十、生产程序 fulltext_fetcher 与规模化实测（141）

在七角度研究基础上,落地了**生产级程序 `fulltext_fetcher/`**:输入 DOI/标题 → 解析元数据 → **13 个免费源回退**定位 OA 全文 → `%PDF` 校验下载入库;具备按域限速+重试退避+坏 host 熔断、并发、断点续跑、结构化日志(attempts/metadata/summary);提供 Python API(`fetch_one/fetch_many/FullTextFetcher`)与子进程 `--print-json` 两种接入,可作为复杂任务的**子模块交接**(见 `接入与交接说明.md`)。

**规模化实测(试点 80 条真实 DOI;语料为 Elsevier/ACS/RSC 等订阅型化学文献)**:
- 成功率 **9/80 = 11.2%**;命中源 Unpaywall 5 / Europe PMC 2 / OpenAIRE 1 / Semantic Scholar 1。
- **47/80 定位到候选,但 38 条下不动**——绝大多数是出版商 **403(付费墙)**;Crossref 候选率最高(48.6%)却几乎全是付费 TDM 链(高噪声、低产出)。
- **结论(印证主线判断)**:免费源对**开放获取/绿色 OA**论文命中良好,对**纯订阅论文**返回"无免费版"是正确边界。**订阅型语料的免费源天花板本就低(经验值 ~10–15%)**;要再高需配 CORE key、加 HTML 落地页内 PDF 解析,或走**角度4 商业服务 / 机构订阅**。
