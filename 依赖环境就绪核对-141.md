## 一、依赖声明 vs 实际 import

### 1.1 已声明（pyproject.toml）

| 包 | 声明位置 | 用途 |
|---|---|---|
| `requests>=2.28` | `[project].dependencies` | HTTP 主路径（http_client、aggregators、多数 sources） |
| `openpyxl` | `[project.optional-dependencies].xlsx` | CLI 读 `.xlsx` 输入 |
| `playwright` | `[project.optional-dependencies].render` | **legacy/备用**；生产 route-B 实际用 **nodriver**，非 playwright |
| `pypdf>=3.17` | `[project.optional-dependencies].qc` | 内容 QC 抽 PDF 正文 |
| `rapidfuzz>=3.0` | `[project.optional-dependencies].qc` | 标题模糊匹配（缺则 difflib 兜底，不 fail-closed） |

`fulltext_fetcher/requirements.txt` 与 `[project].dependencies` 一致，**仅 requests**。

### 1.2 核心路径使用但未声明

| 包 | 使用位置 | 缺省行为 | 严重度 |
|---|---|---|---|
| **`nodriver`** | `render_fetch.py`、`download.py`（route-B/browser_pdf_download）、`browser_search.py`、`tools/flaresolverr_nodriver.py` | ImportError → route-B/FS/browser 路径 **no-op** | **P0**（route-B/MDPI/FS 全不可用） |
| **`curl_cffi`** | `http_client.py`（`FTF_IMPERSONATE_HTTP=1`）、`download.py`、`websearch.py`、`wayback.py` | 降级普通 `requests`；CF/TLS 站命中率降 | **P1**（CF-soft 桶 ROI 降） |
| **`pypdf`** | `download._content_qc_gate`、`tools/qc_content_match.py` | `content_qc_require_deps=True`（默认）→ **拒收不落盘** | **P0**（生产下载几乎全 miss） |
| `httpx` | `fulltext_fetcher/aio.py` | 仅异步路径；主 pipeline 不用 | P3 |
| `websocket-client` | 文档/探针提及 | 非主路径 | P3 |

### 1.3 建议补全（供属主改 pyproject，本文不改）

```toml
[project.optional-dependencies]
browser = ["nodriver>=0.37"]
cf = ["curl_cffi>=0.6"]
full = ["fulltext_fetcher[qc,browser,cf,xlsx]"]
```

或至少在 README/项目索引写清：**生产 = `pip install -e ".[qc]" && pip install nodriver curl_cffi`**。

## 四、缺口与不一致清单

| # | 缺口 | 影响 | 建议 |
|---:|---|---|---|
| 1 | **`nodriver` 未声明** | route-B/FS/MDPI/browser_search 全 no-op，用户不知要装 | 加 `[browser]` extra 或 `full` extra |
| 2 | **`curl_cffi` 未声明** | CF 站 TLS 伪装不可用，websearch/wayback 弱 | 加 `[cf]` extra |
| 3 | **`[qc]` 非默认但 QC fail-closed 默认开** | 只 `pip install -e .` 时下载几乎全被 `content-qc-deps-missing` 拒 | 文档强调必装 `[qc]`；或 dev 默认 `[qc]` |
| 4 | **`playwright` extra 与生产路径不一致** | 用户装 `[render]` 仍无法 route-B（实际要 nodriver） | 注释/playwright 标 legacy；browser extra 指 nodriver |
| 5 | **`requirements.txt` 仅 requests** | 与生产 reality 脱节 | 增 `requirements-prod.txt` 或指向 `pip install -e ".[full]"` |
| 6 | **根 README 无 fulltext_fetcher setup** | 新用户只看 README 找不到主包安装说明 | 链到 `项目结构与功能介绍.md` 或补一节 |
| 7 | **`OPENALEX_KEY` 未在 pyproject/README 主路径强调** | 批量跑 OpenAlex 源静默降级 | 安装后 checklist 第一条 |

## 五、就绪判定（GO / NO-GO）

| 场景 | 依赖 | 环境 | 判定 |
|---|---|---|---|
| 离线 selftest | `requests` | 无 | **GO** |
| 常规免费批量下载 | `requests` + **`pypdf`** + **`OPENALEX_KEY`** + 真实 email | 联网 | **GO**（装 `[qc]`） |

| A5 机构订阅 | 同上 + `--institutional` + cookie/EZproxy | 合法凭据 | **GATE**（无凭据 NO-GO） |
