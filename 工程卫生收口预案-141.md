```text

                publisher_adapter.py cli.py \
                scholar/naming.py run_all.py \
                tools/build_coverage.py                     # coverage 待 -151 广播后
Step 3  python run_all_selftests.py                        # 确认 ALL_SELFTESTS_DONE 全 PASS（含 A5_FRAMEWORK_OK / RENDER_OK）
Step 4  git commit -m "feat: route-B 渲染回收 + A5 机构订阅框架层 + coverage 口径对齐（本波核心码收口）"
Step 5  git add .gitignore                                 # 连同本预案 §二 增量
```
