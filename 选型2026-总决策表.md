## 〇、一页总决策（TL;DR）

按"是否引入外部库"分五档，覆盖全部核心模块：

| 决策档 | 含义 | 落到哪些模块 |
|---|---|---|
| **✅ 采用**（引入库/服务） | 净增量明确、集成成本低 | `curl_cffi`→http_client 反指纹（统一栈）；`byparr`→CF 兜底服务；`DDGS`→websearch 的 DDG 后端；`reCAPTCHA v2 音频+faster-whisper`→captcha 免费实现（默认关） |
| **🔷 整合（可选后端）** | 保留自建主线，作可切换引擎 | `scholarly`→scholar.fetcher 的 GS 原生字段引擎；`primp`→HTTP 反指纹 A/B；`pyalex`→仅"语义找相似" |
| **🧩 借鉴（不整包）** | 移植手法/规则表，自主实现 | `metapub FindIt` 出版商注册表；`paperscraper` 全文回退（BioC-PMC/eLife XML/TDM/bioRxiv S3）；`resp` ACL/ACM 源；`doidownloader` crawl-delay；`camoufox` 受控兜底 |
| **🛠️ 保留自建** | 现有实现已优于所有现成库 | OA 多源发现（20+ 源）；分层 PDF 下载 + `%PDF` 校验；元数据直连（Crossref/OpenAlex/S2/Unpaywall）；命名；chemRxiv 预印本（已自建） |
| **❌ 不用 / 仅登记** | 停更/灰色/被取代/无增量 | 裸 `httpx`、`hrequests`、`Python-Tls-Client` 原版、`undetected-chromedriver`、`playwright_stealth`、`rebrowser-patches`、`FlareSolverr`、`cloudscraper`(降级为轻量首试)、`Buster`、`hcaptcha-challenger`、`PyPaperBot/scidownl/doi2pdf` 接入、`unpywall`、`crossref-commons/crossrefapi`、付费打码 SDK（登记默认关）、镜像站 |

**三条最高优先级行动**（详见 §三 优先级路线）：
1. **P0｜借鉴 `metapub` 出版商注册表**：把 `publisher_adapter.py`/`landing.py` 的出版商 PDF 定位规则从 **7 家扩到数十家** + DOI 前缀 tier 路由表 —— 付费墙 403 高度集中在少数大出版商，**ROI 最高**。
2. **P1｜`curl_cffi` 反指纹统一到 `http_client.py`**：`download/websearch/wayback/scholar.fetcher` **已在用** curl_cffi，唯独主会话 `http_client.py` 仍是裸 `requests`（易被 JA3 判机器人 403）——补一条可选 impersonate 取回路径，直接对症"定位到 OA 却下不下来"。
3. **P1｜接 Wiley/Elsevier TDM API token**：机构学者合规免费，是提升**订阅型语料**命中率最合规的手段（`config.py` 加 key 位、`download.py` 增一档 TDM 尝试）。

## 二、不用 / 仅登记清单（含理由）

| 项目 | 类别 | 结论 | 理由 |
|---|---|---|---|
| 裸 `httpx` | HTTP | ❌ 不用于反指纹 | 无原生 TLS 指纹伪装（JA3 固定），要指纹等于叠 curl_cffi；普通异步 HTTP 用途另说 |
| `hrequests` | HTTP/浏览器 | ❌ 不用 | 2024-12 停更 |

| `tls-client` 系 | TLS | ➖ 备胎 | 指纹最细但轮子重（~40MB Go .so）、BSD-4 广告条款；仅 curl_cffi/primp 都不满足时 |
| `undetected-chromedriver` | 浏览器 | ❌ 不用 | 作者已引导转 nodriver、对现代 Turnstile 渐失效 |
| `playwright_stealth` | 浏览器 | ❌ 不用 | 仅改"声称"、够不到 L3；要补 PW 用 patchright |
| `rebrowser-patches` | 浏览器 | ❌ 不用 | Node-only；Python 侧等价物是 patchright |
| `DrissionPage` | 浏览器 | ⚠️ 慎用 | v4+ 非商用需授权（许可坑） |
| `FlareSolverr` | CF | ➖ 登记/仅老站 | 现代 Turnstile/Managed 已失效、会 hang；byparr 取代 |
| `cloudscraper` | CF | ➖ 轻量首试 | 仅旧式 CF IUAM，对现代 Turnstile 基本无效；作零基建先试 |
| `Buster` | CAPTCHA | ➖ 登记 | 浏览器扩展+人点按钮，不可规模化脚本 |
| `hcaptcha-challenger` | CAPTCHA | ➖ 登记 | 只解 hCaptcha，Scholar 用不到（Scholar=reCAPTCHA） |
| 付费打码 SDK（2captcha/CapSolver/Anti-Captcha） | CAPTCHA | ➖ 登记（默认关） | 持续付费；免费全失败时人工开关（适配器已在 captcha.py） |
| `PyPaperBot` | 下载器 | ➖ 借鉴不接入 | GS+SciHub 中心、易封；批量 UX 我方已覆盖 |
| `scidownl`/`doi2pdf` | 下载器 | ❌ 不用 | SciHub 中心、停更/403 失效、灰色 |
| `unpywall` | OA | ❌ 不用 | 强依赖 pandas、Alpha；我方 Unpaywall 连接器更轻 |
| `crossref-commons`/`crossrefapi` | 元数据 | ❌ 不用 | 维护/社区不及 habanero，且自建已覆盖 |
| `pyalex`/`habanero`/`semanticscholar` | 元数据 | 🔷 参考/可选 | 自建已直连；仅 pyalex 语义相似是增量 |
| 镜像站（烂番薯等） | 下载 | ❌ 不用 | 灰色 + 篡改风险，不宜程序化批量 |
